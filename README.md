# 🏥 Disease Prediction System

A Machine Learning web app that predicts diseases based on symptoms.

## 🚀 Features
- Predict diseases (Flu, Dengue, COVID, Measles)
- Healthy detection system
- Confidence score
- Streamlit UI

## 🧠 Tech Stack
- Python
- Scikit-learn
- Pandas
- Streamlit

## ▶️ Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
